# 🐔 Nano Banana — Pack por Equipo MLB

30 equipos MLB con uniforme oficial + colores + acción sugerida. El workflow n8n rota acciones automáticamente, pero si generas manual en Gemini, copia la variable de tu equipo y úsala dentro del **Paso 2** del master prompt (`nano-banana-pollo-master.md`).

## Cómo usar

1. Abre Gemini → Nano Banana Pro
2. Pega el **Paso 1** del master (una vez por sesión)
3. Pega este template con la variable del equipo:

```
Genera ahora un post vertical 9:16 TikTok @elpollo_apuestas siguiendo MASCOTA/ESTILO/COMPOSICION del paso 1. Ahora el pollo está vestido con el UNIFORME OFICIAL de los {TEAM_NAME} MLB:

UNIFORME = {UNIFORM_DESC}
COLOR_PRIMARIO = {PRIMARY_HEX}
COLOR_SECUNDARIO = {SECONDARY_HEX}
ACCION = {ACTION}
ESTADIO = {STADIUM} con gradas llenas cartoon

Texto del bubble: "¿Quieres ganar hoy con {TEAM_ABBR}?"
Logo del equipo al lado del texto dentro del bubble.
Fondo saturado tono {PRIMARY_HEX} con diamante de béisbol caricaturesco.
Watermark mes "{MES}" amarillo esquina inferior derecha.
```

## Variables rápidas para copy/paste en el workflow n8n

El endpoint `/api/picks-today` ya incluye `pick.colors.primary` y `pick.colors.secondary` por cada pick. El workflow pasa automáticamente:
- `{{ $json.pick.name }}` → nombre completo del equipo
- `{{ $json.pick.abbr }}` → abreviación
- `{{ $json.pick.colors.primary }}` → hex primario
- `{{ $json.pick.colors.secondary }}` → hex secundario

Si quieres editar el prompt automático, el nodo **Build Prompts per Pick** contiene la plantilla. Añade más acciones a `ACTIONS[]` o mejora la descripción del uniforme usando este dossier como fuente.

## Tip avanzado: consistencia de mascota entre posts

Para que el pollo se vea IDÉNTICO entre todos los posts (mismo diseño exacto), una vez generes el primer pollo que te guste:

1. **Descarga esa imagen**
2. En cada prompt siguiente, súbela como **imagen de referencia** (Nano Banana Pro acepta hasta 14 refs)
3. Añade al prompt: *"El pollo debe ser exactamente el mismo personaje que en la imagen de referencia, solo cambia el uniforme y la acción"*

Esto te da el mismo pollo "canónico" en todos los posts, como una mascota real con diseño fijo.
